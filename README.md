# URL Shortener App

Demo URL shortener dengan short link generator, analytics count, dan copy button.

## Features

- Create short link
- Click analytics display
- Copy short URL
- Responsive UI
- Clean SaaS interface

## Tech Stack

- React.js
- Vite
- CSS

## Installation

```bash
npm install
npm run dev
```
